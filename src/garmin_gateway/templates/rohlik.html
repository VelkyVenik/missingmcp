Rohlík MCP Gateway

Connect your Rohlík account to Claude (iOS, Android, Web, Desktop).

What is this?

This gateway lets you do your Rohlík groceries from Claude. It sits in front of Rohlík's hosted MCP server and adds OAuth so you can sign in securely — reorder your usuals, turn a recipe into a cart, check delivery slots, all by asking Claude.

How to connect

  1. In any Claude client, go to Settings → Connectors → Add custom connector (or run claude mcp add --transport http rohlik {PUBLIC_URL}/rohlik/mcp in the CLI).
  2. Set the Server URL to: {PUBLIC_URL}/rohlik/mcp
  3. Claude opens a sign-in page — enter your Rohlík email and password.
  4. Once authenticated, Claude has access to your Rohlík tools. Done!
Important: Rohlík's MCP server requires your email and password with every request, so this gateway stores both, encrypted with AES-256-GCM. They never appear in logs and are sent only to Rohlík. Only use this gateway if you trust the server operator. This instance is operated by {OPERATOR_NAME}{OPERATOR_EMAIL}.

Authentication uses OAuth 2.1 with PKCE. Your credentials are never shared with Claude / Anthropic — they are only used server-side to authenticate with Rohlík. Your groceries & order data is passed through to Claude on demand and is not stored by this gateway.
